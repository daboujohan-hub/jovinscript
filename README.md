## ◉ Origine

Le **Jovïnscript** est né d'un constat simple :

> *L'anglais domine l'informatique. Mais l'anglais est lisible par tout le monde.*
> *Un message sensible envoyé en anglais peut être intercepté, copié, compris.*
> *Il fallait créer quelque chose que personne ne peut lire — même pas un expert.*

Le **19 Mai 2026**, **Omnïval JOHAN** a créé le Jovïnscript — une langue entièrement nouvelle, avec ses propres lettres, sa propre grammaire, ses propres règles. Construite depuis zéro. Sans lien avec aucune langue existante.

Ce n'est pas un chiffrement. Ce n'est pas un code. C'est une **vraie langue** — avec un alphabet de 28 lettres, une grammaire, un vocabulaire, et une application Android complète.

## ✦ Pourquoi le Jovïnscript ?

| Problème | Solution Jovïnscript |
|----------|---------------------|
| Les commentaires de code sont lisibles par tous | Commente en Jovïnscript — personne ne comprend |
| Les messages entre développeurs peuvent être interceptés | Envoie en Jovïnscript — illisible sans la clé |
| L'anglais est obligatoire en informatique | Le Jovïnscript est une alternative neutre |
| Les données sensibles dans le code sont visibles | Masque-les avec le Jovïnscript |
